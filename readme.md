# Métodos Numéricos para Ciencia de Datos
Este repositorio contiene implementaciones y ejemplos prácticos de diversos métodos numéricos aplicados al análisis y procesamiento de datos. Los notebooks incluidos abordan temas como la descomposición en valores singulares (SVD), análisis de componentes principales (PCA), compresión de imágenes y eliminación de fondos en videos.

## Contenido
- `0_implementacion_svd.ipynb`: Implementación de la descomposición en valores singulares (SVD) desde cero, incluyendo la verificación de propiedades de la pseudoinversa de Moore-Penrose.
- `1_pca_usando_svd.ipynb`: Aplicación de PCA utilizando SVD para la reducción de dimensionalidad y visualización de datos. 
- `2_compresion_imagen.ipynb`: Compresión de imágenes mediante SVD, demostrando cómo se pueden reconstruir imágenes con un número reducido de componentes singulares.
- `3_eliminacion_fondo.ipynb`: Eliminación de fondos en videos utilizando técnicas de procesamiento de imágenes y análisis de secuencias de video.
